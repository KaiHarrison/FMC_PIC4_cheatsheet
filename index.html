Setup & install — one flow for ALL hosts
  1. Notify & get approval: Speak with ward staff. Confirm expected downtime and get the green light.
  2. Capture current printers: System Config → Control Panel. Photograph printer name(s) & IP(s).
  3. Insert Media Kit: rear USB.
  4. Reboot PC: press Alt+F4.
  5. Boot from USB: press F9 at POST → choose the Media Kit.
  6. Acknowledge warnings: click OK on two dialogs → install starts (~20 mins). The PC will reboot several times.
  7. When Windows taskbar appears: you may remove the Media Kit and move to the next PC while this one finishes.
  8. On PIC iX Setup screen:
    • Set Hostname & IP per the asset label/table.
    • Subnet: 255.255.248.0; DNS1 172.31.3.1; DNS2 172.31.3.2.
    • Advanced → Domain: intellivue.net.
  9. Reboot after changing network settings.
  10. Printers: Control Panel → add printer(s) with previous name & IP, HP PCL driver, paper size A4.
  11. Run PIC iX Setup:
    • Regional settings: English (Australia).
    • Install Product Databases (wait ~2 mins).
    • Configuration Server → Other Host: SFMCPRIMARY.
    • When asked, password: s735UvC+.
    • Wait ~1 min.
    • Login user philipsfieldengr, password FE+Sup42p0rt!.
    • Request Encryption Configuration (auto).
    • Licensing & topology: leave as-is (assuming Hostname correct).
    • Test speaker function → confirm.
    • Advance to Host Qualification → ensure nothing fails.
    • Finish → select Start Patient Monitoring Services and reboot.
  12. After start: system should auto-connect to monitors.
  13. HL7 & printing:
    • System Config → set HL7 Outbound port per clinical unit mapping below.
    • Set PIC iX Printer Destinations and Monitor Queues (Surveillance only).
