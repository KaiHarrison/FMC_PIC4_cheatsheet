Ryan

Standard install & configuration (applies to every host)
  1. Advise ward staff of brief downtime and obtain OK to proceed.
  2. System Config → Control Panel: photograph existing printer name & IP.
  3. Insert Media Kit (rear USB). Reboot (Alt+F4), then press F9 → boot from USB.
  4. Acknowledge 2 warnings; install runs ~20 min with reboots. When taskbar appears, you may remove the Media Kit and move to the next PC.
  5. PIC iX Setup → set Hostname & IP (per label), Subnet 255.255.248.0, DNS 172.31.3.1 & 172.31.3.2, Domain (Advanced) intellivue.net. Reboot.
  6. Control Panel: install printers (same name & IP), HP PCL driver, paper A4.
  7. Run PIC iX Setup:
    • Region: English (Australia); Install Product Databases (~2 min)
    • Config Server → Other Host: SFMCPRIMARY; Password Vj55UA7=; wait ~1 min
    • When prompted: user philipsfieldengr / password FE+Sup42p0rt!
    • Request Encryption Configuration (auto). Licensing/topology unchanged (check Hostname).
    • Test speaker; proceed to Host Qualification → ensure no failures.
    • Select Start Patient Monitoring Services and reboot.
  8. Confirm system starts and connects to monitors.
  9. System Config: set HL7 Outbound port per clinical unit (see Summary → HL7 table).
  10. Set PIC iX Printer Destinations and Monitor Queues (Surveillance only).
